# Global Boy - Official Artist Website

Professional artist website for Global Boy, featuring his music, biography, and streaming links.

## Features

- ✨ Modern, responsive design with neon aesthetic
- 🎵 Streaming integration (Spotify, Apple Music, YouTube)
- 📱 Mobile-optimized
- 🔍 SEO optimized with structured data

## Getting Started

```bash
pnpm install
pnpm dev
```

## Deploy to Vercel

1. Push to GitHub
2. Go to vercel.com
3. Import this repo
4. Click Deploy

## Technologies

React 19 • Tailwind CSS 4 • TypeScript • Vite
